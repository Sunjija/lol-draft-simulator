# LoL 밴픽 시뮬레이터

개인용 팀게임 밴픽 보조 도구입니다. Riot API key를 입력하면 Riot ID 기반으로 최근 전적/숙련도를 분석해 선수 챔프폭을 갱신하고, 수동으로 입력한 OP.GG/대회 메타 데이터와 조합 규칙을 기반으로 픽/밴 후보를 추천합니다. API 분석 전에는 브라우저 안의 샘플 데이터로 동작합니다.

## 실행

API까지 사용하려면 이 폴더에서 로컬 서버를 실행합니다.

```powershell
python server.py
```

그 다음 브라우저에서 `http://127.0.0.1:8792/`로 접속합니다.

밴픽 화면만 볼 때는 `index.html`을 직접 열어도 되지만, Riot API 분석은 동작하지 않습니다.

## Riot API key 자동 사용

Riot API는 공식 인증 key가 필요해서 key 없이 완전 자동 조회는 불가능합니다. 대신 Setup 화면에서 **이 PC에 키 기억**을 체크하고 `키 적용`을 누르면 다음 실행부터 자동으로 로드됩니다.

저장 위치는 웹으로 열리는 `outputs` 폴더 밖입니다.

```text
work/lol-draft-simulator/riot_api_key.txt
```

키를 지우고 싶으면 Setup 화면의 `저장된 키 삭제`를 누르면 됩니다.

환경변수로 키를 넣고 시작할 수도 있습니다.

```powershell
$env:RIOT_API_KEY="RGAPI-..."
python server.py
```

## 현재 기능

- Riot ID 붙여넣기
- Riot API key 로컬 적용
- ACCOUNT-V1 / MATCH-V5 / CHAMPION-MASTERY-V4 기반 챔프폭 분석
- 블루/레드 사이드 선택
- 하드 피어리스 ON/OFF
- 실제 대회식 20턴 밴픽 순서
- 챔피언 수동 선택
- 추천 후보 클릭 적용
- 상대 픽/밴 예측 후보 클릭 적용
- 선수별 챔프폭 편집 (Player Pool 탭에서 직접 수정 가능)
- 챔피언 초상화 표시 (Data Dragon CDN, Riot API key 불필요 — 서버 실행 중 네트워크로 자동 로드,
  실패 시 이니셜 아이콘으로 자동 대체)
- 전체 173개 챔피언 로스터 (OP.GG 기반 챔피언 티어/라인전 데이터와 직접 정리한 팀게임 밴픽 규칙 반영)
- 라인전 주도권, 바텀 2:2 구도, 정글-미드/탑-정글 시너지, 스왑 밸류, 뚜벅이 응징, 하드 CC/스킬샷 CC 구분 반영
- **랭크/티어 반영**: 선수별 랭크(자동조회 또는 수동입력)를 반영해 (1) 낮은 티어에서 쌓인
  숙련도는 신뢰도를 할인하고, (2) 같은 라인 상대와의 티어 격차를 별도 점수 항목("티어 격차")으로
  추천 가중치에 포함하고, (3) 상대팀 내 유독 티어가 높은 선수의 챔프폭 쪽으로 밴 추천 우선순위를
  높입니다. 랭크는 Setup 탭에서 직접 선택하거나, Riot 팀 분석 시 LEAGUE-V4로 자동 조회됩니다.
- 현재 픽을 피어리스 사용 기록에 추가
- 실전 타이머와 로컬 BGM 재생
- 로컬 저장/불러오기
